# 10 Networking Commands Reference

## 1. Cisco IOS Commands

### Category: Configuration Commands  
- **Command**: `configure terminal`  
- **Use Case**: To enter configuration mode for device settings.

### Category: Show Commands  
- **Command**: `show ip interface brief`  
- **Use Case**: To display a summary of the interface status.

## 2. Linux Networking Commands

### Category: Network Configuration  
- **Command**: `ifconfig`  
- **Use Case**: To display or configure network interfaces.

### Category: Network Testing  
- **Command**: `ping`  
- **Use Case**: To test the reachability of a host.

## 3. Windows Networking Commands

### Category: Network Configuration  
- **Command**: `ipconfig`  
- **Use Case**: To display or configure IP address settings.

### Category: Network Troubleshooting  
- **Command**: `tracert`  
- **Use Case**: To trace the route of packets to a network host.
